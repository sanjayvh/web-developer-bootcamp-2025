HTML

HyperText Markup Language is the standard language used to create and structure web pages. It uses tags and elements to define headings, paragraphs, images, links, and other components that browsers display on the screen.

It is a markup language, not a programming language.

This means it annotates text to define how it is structured and displayed by browsers
